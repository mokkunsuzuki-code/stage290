Verification URL UI

Stage289 の Verification API を、人が見て判断しやすい検証URL画面にしたものです。 URL と manifest を入力すると、判定・信頼スコア・内訳をブラウザで確認できます。

この画面の役割

  • 検証ロジックそのものは Stage289 API が担当
  • Stage290 は「見せる層」として結果を整理
  • ターミナルを使わずに検証できる
  • 将来は共有URLや証明ページへ発展可能

検証フォーム

URL と manifest JSON を入力して検証を実行します。 初期値には accept になるサンプルを入れています。
